short explanation of files:

algorithm files - 

CompareSum.py - main file of the algorithm, gets the input, calculates using all the different modules and returns the output
GoogleFunctions.py - modules related to the Google search results of the input
PagesFunctions - modules related to inside-website analysis of the Goggle search results

GUI files - 

main.py - main file of GUI and calls for the algorithm (CompareSum.py)
comparesum_kivy.kv - style and design of GUI, written in kv language (similiar to CSS)

other - 
ttf files - fonts for the GUI
png and jpg and simple_images folder - images for the GUI
txt files - saved inputs to use in the app without using the API (because of query limit)
